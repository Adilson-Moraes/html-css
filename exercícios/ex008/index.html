Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag<b>, essa tag é não semântica, pois está focada somente na forma.

Nessa frase temos um termo em destaque usando a tag <strong>, que é uma tag semântica, pois está tornando as palavras "destacadas".

Itálico / Ênfase

Nesta frase, temos um termo em itálico, usando a tag <i> que é uma tag não semântica, pois está focada em forma.

Nesta frase temos um termo em enfâse usando a tag <em> que é uma tag semântica pois está dando "ênfase" as palavras.

Texto Marcado

Podemos usar também um texto marcado usando a tag <mark>.

Texto grande e pequeno

Estamos criando um texto grande usando a tag <big> e um texto pequeno usando a tag <small>.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Texto Sobrescrito

Utilizamos a tag <sup> para fazer um texto sobrescrito, como x20+3.

Texto Subscrito

Utilizamos a tag <sub> para fazer um texto subscrito como H2O.
